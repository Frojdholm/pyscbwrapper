# pyscbwrapper
Version 0.1

Python wrapper for SCB's API, which can be found at http://www.scb.se/en/api.
Updated version compatible with the changes in the API is coming soon. 

Dependencies: requests>=2.12.4

To import: 
```python
from pyscbwrapper import SCB
```
